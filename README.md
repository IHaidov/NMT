# Тренажер НМТ з математики (Україна)

Це простий веб-додаток для тренування варіантів НМТ з математики у стилі Moodle. Питання беруться з файлу `db.json`, формули відображаються за допомогою LaTeX (MathJax), передбачено три типи завдань:

- вибір однієї правильної відповіді;
- завдання на встановлення відповідності;
- завдання з короткою числовою відповіддю.

## Запуск

**Важливо:** Завдання завантажуються з файлу `db.json`. Браузер блокує такий запит при відкритті сторінки через `file://`. Потрібно запустити локальний сервер:

1. Відкрийте термінал у папці проєкту і виконайте: **`npm start`** (потрібен [Node.js](https://nodejs.org/)).  
   Альтернатива без Node: **`python -m http.server 3000`** (потім відкрийте http://localhost:3000).
2. У браузері відкрийте: **http://localhost:3000**
3. Уведіть ПІБ та натисніть **«Почати тест»**.
4. Для перегляду джерел завдань скористайтеся посиланням «Джерела завдань» у шапці або відкрийте `resources.html`.

## Структура

- `index.html` — головна сторінка з тестом.
- `resources.html` — сторінка з описом використаних збірників та підручників.
- `styles.css` — стилі інтерфейсу (дружній, у дусі Moodle).
- `app.js` — логіка вибору випадкових неповторюваних завдань, навігація, підрахунок балів та відсотків.
- `db.json` — база завдань (LaTeX, зображення, правильні відповіді).
- `herb.svg` — герб України (можна замінити на іконку з [Flaticon — Kalashnyk](https://www.flaticon.com/ru/free-icons/)).
- `server.js` — простий HTTP-сервер для коректного завантаження `db.json`.

## Оцінювання

- 15 завдань з вибором однієї відповіді по 1 балу;
- 3 завдання на встановлення відповідності (0–3 бали);
- 4 завдання з короткою відповіддю (0 або 2 бали).

Максимальний бал — 32, також обчислюється відсоток результату.
